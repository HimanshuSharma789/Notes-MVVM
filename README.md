# Notes-MVVM
This app allows user to create personal notes and save them. Build using MVVM architecture on Kotlin.

### Features of app:
* Dark theme support
* Allows user to create note
* Delete notes
* Update
* Search feature
* Pin Notes to the Top
* Adjust background color
* Share your notes


### Components involved:
* ViewModel
* LiveData
* Databinding
* Room Database
* RecyclerView
* BindingAdapter
